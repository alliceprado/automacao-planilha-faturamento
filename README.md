# automacao-planilha-faturamento
Projeto de Automação de processos administrativos e dashboard para otimização de fluxo de trabalho para TCC de MBA em Data Science e Analytics

Este repositório contém a solução desenvolvida para o TCC do MBA em Data Science e Analytics da USP/ESALQ.

Funcionalidades:

Automação: Redução do processamento manual de 180 para 5 minutos.
Distribuição: Divisão equitativa de demandas entre analistas ativos.
Consolidação: Base unificada para alimentação de Dashboard em Power BI.
